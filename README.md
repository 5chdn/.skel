# .skel
my (arch) linux default skel of dot files

```bash
$ https://github.com/5chdn/.skel.git
$ rm -rf .skel/.git/ .skel/LICENSE .skel/README.md
$ rsync .skel/* $HOME/
$ source $HOME/.zshrc
```
